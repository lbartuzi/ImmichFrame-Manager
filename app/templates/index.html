Add account

Each account maps to one Immich account/API key in ImmichFrame. For clean album management, create a separate Immich user such as frame.

How the album modes behave

Manual

Do not overwrite album lists. Useful while testing or when you want direct UUID control.

Allow all

Every cached Immich album that passes filters is written to ImmichFrame.

Show selected

Only checked albums are written. This is the safest strict allow-list mode.

Hide selected

Everything matching the filters is written except checked albums.
